Fiscal policy shocks and stock prices

Author

Hanwen Zhang

Abstract. This is a reserarch proposal, the research is about measuring fiscal policy effect using Structural VAR models in Australian Economy.

Keywords. fiscal policy shock, SVAR, tax shocks, stock price

Introduction

How does tax cuts affect the economy? Will tax reductions boost stock market? This report aims to answer these questions in the background of Australian Economy. These questions are crucial to understanding of dynamic economics and the stimulus policy-making process.

Literature Review

Methodology

Structural vector autoregression (SVAR) models can be used to estimate the impact of taxation shocks in Australian economy. The benchmark model is defined as: \[\begin{align} B_0Y_t = b_0 + \sum_{i=1}^{p}B_{i}Y_{t-i}+u_t \end{align}\] \[\begin{align} u_t|Y_{t-1}\sim iid(0_N,I_N) \end{align}\]

\(Y_t\) is \(N\times1\) matrix of endogeneius variables, N represents the number of endogeneous variables and p is the number of lags, \(B_0\) is \(N\times N\) matrix capturing contemporaneous relationships between variables, and \(u_t\) is \(N\times1\) vector conditionally on \(Y_{t-1}\) orthogonal structural shocks.

The reduced form representation is: \[\begin{align} Y_t = \mu_0 + \sum_{i=1}^{p}A_{i}Y_{t-i}+\epsilon_t \end{align}\] \[\begin{align} \epsilon_t|Y_{t-1}\sim iid(0_N,\Sigma) \end{align}\]

The covariance matrix of \(\epsilon_t\) can be written as: \[\begin{align} \Sigma=B_0^{-1}{B_0^{-1}}^{'} \end{align}\]
